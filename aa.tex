\section{What Was Implemented}
This document explains, in detail, the full implementation completed in this repository for the options-based equity return research workflow:
\begin{enumerate}[leftmargin=1.2em]
  \item multi-ticker data acquisition and completion,
  \item full feature engineering (stock + options + macro + event),
  \item panel preprocessing with leakage-safe temporal design,
  \item multi-architecture model suite (XGBoost, FFNN, TabNet, TFT),
  \item explainability outputs (XGB SHAP, FFNN SHAP, TabNet masks, TFT training/eval),
  \item ``future plans'' implementation (rolling windows, regime analysis, significance tests, conformal, SHAP interaction/local/bootstrap CI, consensus analysis).
\end{enumerate}

\subsection{Raw Options and Underlying Data}
Primary source: \texttt{philippdubach/options-data} parquet mirrors, stored as:
\begin{itemize}[leftmargin=1.2em]
  \item \texttt{data/raw/options\_data\_repo/<ticker>/options.parquet}

\end{itemize}

\section{Feature Engineering}
Script: \texttt{build\_panel\_features\_with\_vix.py}

\subsection{Computed Features}
\begin{itemize}[leftmargin=1.2em]
  \item \textbf{Options features} (\texttt{feat\_01}--\texttt{feat\_26}): put/call ratios, IV signals, activity/microstructure, gamma/charm exposures, 0DTE, implied skew/kurtosis proxy.
  \item \textbf{Stock/macro controls} (\texttt{feat\_27}--\texttt{feat\_42}): returns, HV, RSI, MACD, Bollinger, ATR, VIX controls, rates, sector ETF returns.
  \item \textbf{Event/temporal} (\texttt{feat\_43}--\texttt{feat\_48}): days to earnings/dividend/OpEx, earnings flag, weekday, quarter-end.
  \item \texttt{feat\_49} sentiment is intentionally left NaN (excluded).
\end{itemize}

\subsection{Targets}

target\_r\_{t+1}=\frac{P_{t+1}}{P_t}-1,\quad

\section{Preprocessing}
Script: \texttt{preprocess\_panel\_training\_data.py}

\subsection{Operations}
\begin{enumerate}[leftmargin=1.2em]
  \item numeric coercion and inf cleanup,
  \item anomaly cleanup for bounded features (spread/IV/rank/0DTE),
  \item target+ID filtering,
  \item temporal split by date (60/20/20) before imputation,
  \item train-only winsorization (0.1\%, 99.9\%),
  \item train-only median imputation (ticker then global, then 0 fallback),
  \item final drop if any required column still NaN.
\end{enumerate}

\section{Model Suite Implementation}
Script: \texttt{train\_panel\_model\_suite.py}

\subsection{Architectures}
\begin{itemize}[leftmargin=1.2em]
  \item FFNN: Dense 256-128-64-32 with ReLU + Dropout.
  \item XGBoost: classifier+regressor.
  \item TabNet: classifier+regressor with mask-based attention.
  \item TFT: one-step horizon, 20-day lookback, static categorical ticker/sector.
\end{itemize}
