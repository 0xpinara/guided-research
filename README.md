# guided-research

Guided-research project by **Pınar Aksoy** (METU, Computer Engineering).

**How much do option metrics predict short-horizon stock returns?**
A power- and cost-aware evaluation, with a volatility positive control.

The full study — code, the embargoed walk-forward evaluation, the multiple-testing
corrections, the volatility positive control, the power analysis, and the saved
result tables that reproduce every number and figure in the paper — lives in:

### → [`guided/options_research/`](guided/options_research/)

See that folder's [README](guided/options_research/README.md) for the headline
result, repository layout, install instructions, and a table mapping each reported
number to the script that produces it.
